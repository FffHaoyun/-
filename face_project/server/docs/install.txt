安装依赖包

后端：tornado
图像处理：opencv-python
数据库链接驱动：pymongo
表单验证：wtforms
工具集：werkzeug

卸载依赖包
pip uninstall -r requirements.txt

使用默认镜像源安装
pip install -r requirements.txt

加速国内镜像源安装
pip install -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com -r requirements.txt

查看依赖包版本
pip list
pip freeze
